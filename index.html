// 课程体系

不是又一套动作，
是一套看懂身体的系统。

市面上多数体态课在教"这个问题练哪个动作"。这门课教你的是背后那套逻辑——综合四套国际前沿体系，让你面对任何一个身体，都能自己评估、自己判断、自己开方案。

DNS
捷克发育神经学

APTA
美国物理治疗

运动康复的底层逻辑，临床康复视角下的训练设计。

PRI
姿势恢复研究所

专门处理高低肩、长短腿、骨盆回旋等不对称问题。

松动

从关节层面解除卡压与受限，而不只是从肌肉入手。

整

闭

评估·手法·训练闭环

先评估找到原因，再用松解手法解除卡压，最后用对症训练重建——一条完整闭环。

// 底层逻辑

体态是结果，
不是原因。

所有体态问题都能沿着同一条链条往回推。看懂这条链，你才知道该在哪里下手——而不是对着外形硬掰。

行为→ 姿势→ 结构→ 体态（外在呈现）
举例 · 悬垂腹：久坐含胸看手机 → 胸廓塌陷呼吸短浅 → 骨盆前倾、腹内压往前走 → 悬垂腹。你要找的，是中间那个"他到底做错了什么"。
01

筋膜张拉结构

从头到脚被筋膜包成一张蜘蛛网，任一处受力全网形变。这是"牵一发动全身"整体观的底层逻辑。

02

胸廓 / 骨盆对位

胸廓与骨盆的相对位置，直接决定腹部形态与脊柱体态。对位错了，怎么练腹都没用。

03

腹内压（IAP）

膈肌 + 盆底 + 腹横肌构成的内核心。腹内压的方向决定了你是收腹还是顶腹，是核心中的核心。

04

人体不对称理论

人天生不对称。右侧承重模式带来全身连锁反应——PRI 视角让你看懂长短腿、骨盆回旋的真正成因。

// 四大理论板块

每个动作背后，
都有一套说得清的理论。

理论不是背书，是你以后独立判断、给客户解释的底气。课程理论分为基础、中轴、下肢、上肢四个板块——这套认知体系，正是你和大多数培训师最大的差距来源。

壹

基础理论 · 看身体的底层语言

用五大学科 + 四套国际体系，建立"整体观"与"对抗重力"这条贯穿全程的主线。先有共同语言，后面每个动作才讲得通。

生物力学神经科学动态张拉整体解剖 · 物理DNS / APTA / PRI / 关节松动行为→姿势→结构→体态三维平面 · 三种力关节联动 · 重心转换调整顺序：中轴→四肢→整体

蜘蛛网：从任意一点施力，整张网都有响应。没有局部问题，只有整体失衡在某处的体现。

贰

中轴理论 · 决定一切的地基

胸廓、骨盆、脊柱三者的对位，决定了腹内压、重心和四肢的起始角度。中轴没对位，加再多核心训练都是代偿。

腹内压双层容器外层骨性 + 内层肌肉胸廓弹性矢状面垂直对位三平面参与的核心胸廓恢复弹性骨盆恢复旋转两者恢复交互

保温杯：盖子歪了、底座裂了，再使劲拧也密封不住。看到客户"核心无力"，先问的不是肌肉强不强，而是"容器对位了吗"。

叁

下肢理论 · 从骨盆螺旋到足底

下肢不是四个独立零件，而是一个从骨盆出发、一路螺旋到足底的完整系统。每一段反向扭转，整条腿才有张力、才能承重。

下肢螺旋排列骨盆运动规律：旋前→外展外旋髋 · 股骨 · 膝 · 胫骨 · 足踝髋膝踝动力链妈妈臀假胯宽X 型腿 · O 型腿

前轮定位：定位歪了，换轮胎、换刹车片都没用——每次都在歪的方向上磨损。膝盖内扣，根源可能在骨盆、在髋、在踝。

肆

上肢理论 · 肩颈问题的真正源头

颈椎病、富贵包、圆肩驼背、肩袖损伤——大多数根源不在肩颈，在中轴。上肢是从脊柱曲度一路传导到肱骨头的排列链。

脊柱 S 型三段曲度胸椎 = 旋转核心 & 最常见断点肩胛骨贴合胸廓翼状肩胛圆肩驼背 · 头前引富贵包 · 颈椎病

多米诺骨牌：第一张倒了，后面一路倒。很多人只去扶最后一张——颈椎、肩膀，却忘了找第一张是哪里倒的。

认知的高度，决定了解决问题的能力。 // 这门课的每个动作和理论，都有五个学科作为支撑，而不是凭经验摸索
// 六天课程详解

六天，逐日带你打通全身。

从核心地基（腹内压、腹部、悬垂腹），到不对称与骨盆，到下肢腿型与足部，再到上肢动力链。点开每一天，看看具体讲什么。

// 为什么评估必须先行
没评估就开练，
像 没诊断就开药。 同样是悬垂腹，有人是高张腹、有人是骨盆前移、有人是小肋骨角型——成因不同，方案完全不同。
01

先评估，找到真正成因

三平面观察、杆子测试、奥博测试、肋骨角、假胯宽三步评估、足底功能测试——一套可复制的评估流程。

02

再松解，解除卡压

腹部擀皮、膈肌 / 盆底松解、胸腰筋膜松解、跟骨巨骨复位、髋 / 肩 / 足踝关节松动术。

03

后训练，对症重建

9090 系列、不对称训练、平衡训练、臀部塑形进阶——每个体态问题都配一套完整可落地的训练方案。
